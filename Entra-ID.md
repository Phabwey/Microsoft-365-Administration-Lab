# Microsoft Entra ID Administration

## Overview

Microsoft Entra ID is Microsoft's cloud-based identity and access management service. This lab demonstrates how users, groups, permissions, and authentication methods are managed within Microsoft 365.

---

## User Management

The following user administration tasks were performed:

- Created user accounts.
- Updated user information.
- Assigned departments and job titles.
- Enabled and disabled user accounts.
- Reset passwords.

---

## Group Management

The following groups were created:

| Group Name | Purpose |
|------------|----------|
| IT Support Team | Technical support |
| Finance Team | Financial operations |
| HR Team | Human resource management |
| Sales Team | Sales and customer engagement |

Users were assigned to groups according to their roles and responsibilities.

---

## Administrative Roles

The following Microsoft Entra roles were configured:

- Global Administrator
- User Administrator
- Helpdesk Administrator
- Security Administrator
- Teams Administrator
- Exchange Administrator

---

## Authentication Methods

The following authentication methods were configured:

- Password authentication
- Microsoft Authenticator
- SMS verification
- Email verification

---

## Identity and Access Management

The following identity and access tasks were completed:

- Assigned role-based access permissions.
- Configured authentication settings.
- Managed user privileges.
- Reviewed user sign-in activity.

---

## Security Features

The following security measures were implemented:

- Multi-Factor Authentication (MFA)
- Password reset policies
- User access reviews
- Account monitoring
