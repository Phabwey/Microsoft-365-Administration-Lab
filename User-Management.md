# User Management

## Overview

User management is one of the primary responsibilities of Microsoft 365 administrators. This lab demonstrates the creation and management of users, licenses, permissions, and administrative roles within Microsoft 365.

---

## User Accounts

The following user accounts were created:

| User | Department | Role |
|-------|-------|-------|
| paballo.litabe@company.com | IT Support | Technician |
| john.smith@company.com | Finance | Accountant |
| sarah.jones@company.com | Human Resources | HR Manager |
| david.brown@company.com | Sales | Sales Representative |

---

## User Configuration

The following settings were configured:

- Display name
- User principal name (UPN)
- Department information
- Job title
- Contact details
- Account status

---

## License Assignment

The following Microsoft 365 licenses were assigned:

| License | Purpose |
|----------|----------|
| Microsoft 365 Business Basic | Email and collaboration |
| Microsoft Teams | Communication |
| Exchange Online | Email management |
| SharePoint Online | File sharing |

## Administrative Roles

The following administrative roles were configured:

- Global Administrator
- User Administrator
- Helpdesk Administrator

## Password Management

The following tasks were performed:

- Reset user passwords.
- Forced password changes.
- Unblocked users.
- Enabled and disabled accounts.
- Updated account information.
