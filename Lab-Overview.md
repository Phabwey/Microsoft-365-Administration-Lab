# Microsoft 365 Administration Lab Overview

## Introduction

This lab simulates a Microsoft 365 enterprise environment and demonstrates the administrative tasks performed by IT Support Specialists and Junior System Administrators.

The project focuses on Microsoft Entra ID, user and license management, Microsoft Teams administration, Exchange Online, Multi-Factor Authentication (MFA), and security policies.

---

## Lab Objectives

- Create and manage Microsoft 365 users.
- Configure Microsoft Entra ID.
- Assign licenses and permissions.
- Manage Microsoft Teams.
- Configure Exchange Online.
- Enable Multi-Factor Authentication (MFA).
- Apply security policies.
- Troubleshoot user access issues.

---

## Lab Components

### Microsoft Entra ID

Microsoft Entra ID provides identity and access management for users, groups, and applications within the organization.

### Microsoft 365 Admin Center

The Admin Center is used to manage users, licenses, billing, and organizational settings.

### Microsoft Teams

Microsoft Teams provides communication and collaboration capabilities across departments.

### Exchange Online

Exchange Online manages email services, mailboxes, and communication policies.
